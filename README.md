# Dashboard Prediksi Data

Proyek ini merupakan tugas mata kuliah Data Mining yang bertujuan untuk melakukan pengolahan, analisis, dan prediksi data serta menampilkan hasilnya dalam bentuk dashboard interaktif.

## Tools yang Digunakan
- Python
- Google Colab
- Streamlit

## Fitur
- Pengolahan data
- Model prediksi
- Visualisasi data dalam bentuk dashboard

## Keterangan
Proyek ini dikembangkan sebagai bagian dari pembelajaran mata kuliah Data Mining.
